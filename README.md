# Carmen's Portfolio

A personal portfolio built with React and Vite to showcase my profile, bootcamp projects, and freelance work.

## Tech Stack

- React
- Vite
- React Router
- CSS

## Features

- About page
- Bootcamp projects section
- Freelance work section
- Custom 404 page for unknown routes
- Responsive layout

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
